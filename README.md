## LumenBuddy - Emotional Support AI Chatbot
It offers both offline and online option for users to express their emotion in a judgement-free environment. Users can receive friendly and supportive feedback from AI. 
Besides from chatting, it provides:
1. Mood Evaluation & Tracking
    - Detects emotional cues during conversations
    - Generates a daily Emotion Index (scale 1–5)
    - Tracks patterns with a mood diary
2. Therapeutic Games
    - Interactive games for emotional regulation and stress relief
    - Customized based on your Emotion Index
3. Relaxation Tools
    - Offers soothing music and guided breathing exercises

Offline Technical Setup:
1. Environment Preparation
Download and install GPT4All (Llama 3.2 1B Instruct model) from nomic.ai locally
Develop the system using Visual Studio

2. Flash Player Support
Since Adobe Flash is no longer supported on Windows 11, a Flash player is needed to enable Flash-based therapeutic games

Notes: This repository stores the source code only. Executable files and installation packages are not included.
